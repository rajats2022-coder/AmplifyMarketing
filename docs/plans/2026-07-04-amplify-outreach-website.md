# Amplify Outreach Website Implementation Plan

> For Codex: build the site directly in this client folder using the static S4 client-site pattern.

**Goal:** Create a complete static website for Amplify Outreach with strong conversion copy, multi-page SEO coverage, and local preview tooling.

**Architecture:** Use standalone HTML pages with shared `assets/styles.css` and `assets/script.js`, copied logo imagery, and Node helper scripts for localhost serving and screenshots. Keep forms client-side until the final email or CRM destination is supplied.

**Tech Stack:** Plain HTML, CSS, JavaScript, Tailwind-free static assets, Google Fonts, Lucide icons, Node HTTP server, Chrome CDP screenshot scripts.

---

## Tasks

1. Create assets and install the provided logo.
2. Write the brand brief and local preview documentation.
3. Build shared CSS and JavaScript for layout, navigation, FAQ, forms, reveal animation, and chatbot.
4. Create the homepage, service pages, pricing page, results page, industry SEO pages, and contact page.
5. Add `llms.txt`, `robots.txt`, and `sitemap.xml`.
6. Run the local server on port 3004.
7. Capture desktop and mobile screenshots.
8. Fix visual issues found in screenshots.
9. Re-run desktop and mobile screenshots for the second verification round.
